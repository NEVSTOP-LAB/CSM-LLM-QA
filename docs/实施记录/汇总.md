## 遇到的问题与解决

### 1. Markdown 分块 heading 命名不一致
- **问题**: 无标题文档的 heading 被设为 "Untitled" 而非 "Document"
- **解决**: 修正测试期望值为 "Untitled"

### 2. BudgetExceededError 重复告警
- **问题**: `process_article()` 和 `run_articles()` 都触发了预算告警
- **解决**: `process_article()` 只 raise，`run_articles()` 统一处理告警

### 3. run_articles() 方法缺失
- **问题**: 测试调用 `runner.run_articles()` 但方法不存在
- **解决**: 从 `run()` 中提取 `run_articles()` 方法，便于测试
